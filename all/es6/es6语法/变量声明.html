• let
  •     	(function () {
    	    for (var i = 0; i < 3; i++) {
    	        console.log(i)
    	    }
    	    console.log(i)
    	})();

    	输出：0,1,2
    	输出：3
        
        	(function () {
    	    for (let i = 0; i < 3; i++) {
    	        console.log(i)
    	    }
    	    console.log(i)
    	})();

    	输出：0,1,2
    	输出：Uncaught ReferenceError: i is not defined.未捕获引用错误：未定义i
		
        	let a = 0;
    	let a = 1;

    	输出：Uncaught SyntaxError: Identifier 'a' has already been declared.未捕获语法错误：标识符“a”已声明。
    1、在es6中每一对{}都是一个块作用域，而let声明的变量只能在它声明的块作用域使用
    2、在es6中默认开启strict（严格的）代码模式，所以报错是Uncaught ReferenceError: i is not defined而不是直接报错undefined
    3、let不能对同一个变量重复命名
  • const
  •     	function () {
    		const PI = 3.1415926;
    		PI = 3.14;
    		console.log(const)
    	}

    	输出：Uncaught TypeError: Assignment to constant variable.未捕获的类型错误：分配给常量变量。
    	
        	(function () {
    		const PI;
    		PI = 3.14;
    		console.log(PI);

    		const test = {
    			a : 1,
    		}
    		test.a = 2;
    		test.b = 3;
    		console.log(test)
    	})();

    	输出：Uncaught SyntaxError: Missing initializer in const declaration未捕获语法错误：常量声明中缺少初始值设定项
    	输出：{a: 2, b: 3}
    1、在es6中每一对{}都是一个块作用域，而const声明的变量只能在它声明的块作用域使用
    2、const声明的是常量，只能读不能改(consts声明的对象除外)
    3、const声明的时候必须赋值，不能先声明后赋值
