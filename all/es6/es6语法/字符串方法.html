  • 二、对Unicode支持：codePointAt()、String.fromCodePoint()
  • 背景：在ES5中，我们经常使用 charAt(num) 方法返回指定位置的字符，用 charCodeAt(num) 方法返回指定位置的字符的 Unicode 编码。这个返回值是 0 - 65535 之间的整数。
    在JavaScript内部，字符以 UTF-16 的形式存储，每个字符固定为2个字节，对于那些需要4个字节来储存的字符Javascript认为它们是2个字符。
    此时 charAt() 方法和 charCodeAt() 方法都无法完整获取这样的字符和它的 Unicode 编码。
    因此，ES6使用 codePointAt() 方法来支持存储4字节的字符。 
  •         {
            let str = '𠮷';
            console.log(str.length);
            输出：2

            console.log(str.charAt(0));
            输出：�

            console.log(str.charAt(1));
            输出：�

            console.log(str.charCodeAt(0));
            输出：55362

            console.log(str.charCodeAt(1));
            输出：57271

            console.log(str.charCodeAt(0).toString(16));
            输出：d842
            console.log(str.charCodeAt(1).toString(16));
            输出：dfb7
            console.log('\ud842\udfb7')
            输出：𠮷


            console.log(str.codePointAt(0));
            输出：134071

            console.log(str.codePointAt(1));取后面两个字节的值，和str.charCodeAt(1)取到的值一样
            输出：57271
        }

        {

  • 四、includes()、startsWith()、endsWith()：判断字符串是否包含在另一个字符串中
  •         {
            let str = "string";
            console.log('includes', str.includes("aaa"));//返回布尔值，表示是否找到了参数字符串。
            输出：false

            console.log('start', str.startsWith('str'));//返回布尔值，表示参数字符串是否在源字符串的头部。
            输出：true

            console.log('end', str.endsWith('ng'));//返回布尔值，表示参数字符串是否在源字符串的尾部。
            输出：true
        }

        接受第二个参数，可选的，表示开始搜索的位置
        {

            输出：true

            console.log(str.endsWith("an", 10));
            输出：true

            console.log(str.includes("an", 11));
            输出：true

        }
        注意：使用第二个参数n时，endsWith() 指的是针对前n个字符

    实例：模板编译 
            {
            var template = `
            <ul><% for(var i=0; i < data.supplies.length; i++) { %>
                <li><%= data.supplies[i] %></li><% } %>
            </ul>`;

            function compile(template) {
                var evalExpr = /<%=(.+?)%>/g;
                var expr = /<%([\s\S]+?)%>/g;

                template = template.replace(evalExpr, '`); \n  echo( $1 ); \n  echo(`').replace(expr, '`); \n $1 \n  echo(`');
                template = 'echo(`' + template + '`);';

                var script = `(function parse(data){
                var output = "";

                function echo(html){
                  output += html;
                }

                ${ template }

                return output;
              })`;

                return script;
            }
            var parse = eval(compile(template));
            console.log(parse({ supplies: ["Ethan", "Hello", "World"] }));

            输出：
            
                <ul>
                    <li>Ethan</li>
                    <li>Hello</li>
                    <li>World</li>
                </ul>
            

            这里就是使用正则表达式将template模板转换成了类似如下代码并以字符串形式返回，然后使用eval方法把它变成可执行的。
            echo('<ul>');
            for(var i=0; i < data.supplies.length; i++) {
              echo('<li>');
              echo(data.supplies[i]);
              echo('</li>');
            };
            echo('</ul>');
        }

  • 八、标签模板：这是函数调用的一种特殊形式，标签指的就是函数，紧跟在后的模板字符串就是它的参数。第一个参数是被插入值分割成数组的字符串，后面就依次是插入值了。

                name: 'Ethan ',
                info: 'Hello World'

            function abc(s, v1, v2) {
                console.log(s);

    实例：过滤HTML字符串， 以防用户输入恶意内容。 
            {
            var sender = '<i>Tom</i>';
            var message = SaferHTML `<p>${sender} has sent you a message.</p>`;
            console.log(message);
            输出：<p>&lt;i&gt;Tom&lt;/i&gt; has sent you a message.</p>

            function SaferHTML(templateData) {
                var str = templateData[0];
                for (var i = 1; i < arguments.length; i++) {
                    var arg = String(arguments[i]);

                    // 转义替换中的特殊字符。
                    str += arg.replace(/&/g, "&amp;").replace(/</g, "&lt;").replace(/>/g, "&gt;");

                    // 不要转义模板中的特殊字符。
                    str += templateData[i];
                }
                
                return str;
            }
        }
  • 九、String.raw()：唯一一个内置的模板字符串标签函数，是用来获取一个模板字符串的原始字面量值的。
  • 1、语法 
            String.raw`templateString`; 

        String.raw(obj, ...substitutions);
  • 2、参数 
            templateString： 必需，模板字符串
        {
            console.log(String.raw `Hi\n${1+2}`);
            输出：Hi\n3

            console.log(`Hi\n${1+2}`);
            输出：
                    Hi
                    3
                 
        }

        obj： 必需，一个模板字符串的“调用点对象”。类似{ raw: ['foo', 'bar', 'baz'] }
        ...substitutions： 可选，任意个可选的参数，表示任意个内插表达式对应的值
        {
            console.log(String.raw({ raw: 'test' }, 1, 2, 3, 4));
            输出：t1e2s3t

            console.log(String.raw({ raw: ['t', 'e', 's', 't'] }, 1, 2, 3, 4));
            输出：t1e2s3t
            
            原理
            String.raw = function(strings, ...values) {
                var output = "";
                for (var index = 0; index < strings.length; index++) {
                    output += strings.raw[index] + values[index];
                }
                output += strings.raw[index]
                return output;
            }
        }
  • 3、返回：给定模板字符串的原始字面量值。
  • 4、异常：如果第一个参数没有传入一个格式良好的调用点对象，则会抛出 TypeError 异常。
  • 5、妙用：我们经常会需要在js文件中书写 html 或者 css。在这种场景下，使用模板字符串是个很好的方式，这样就不需要进行复杂而丑陋的字符串拼接了。 
            {
            var html = String.raw;
            html = `
            <div class='bg-blue'>
                <p>哈哈哈</p>
            </div>`
        }
