• overflow-基本属性
  • visible：默认值，内容溢出容器（在ie6下: 内容不会溢出容器而是容器被撑开）
    hidden：隐藏超出部分
    scroll：产生滚动条
    auto：内容超出容器产生滚动条，不超出则不会产生滚动条
    inherit：不常用。
  • overflow-基本特性
  • 当overflow-x与overflow-y值相同时，等同于overflow
    当overflow-x与overflow-y值不相同时，且其中一个值为visible，另一个被赋予：hidden auto scroll，那这个visvible会被重置为auto。
  • overflow: 作用前提
  • 1、非 display: inline 水平
  • 2、对应方位的尺寸限制: width/height/max-width/max-height/absolute 拉伸
  • 3、对于单元格 td 等, 需要 table 为 table-layout: fixed 状态才可以
  • overflow:visible
  • 在IE7浏览器下，文字越多，按钮两侧padding留白就越大，IE8+则是正常的
  • IE7的这个问题可以用overflow:visible解决
