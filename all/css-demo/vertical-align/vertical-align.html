• vertical-align起作用的前提
  • 1、vertical-align只能应用于inline、inline-block水平及table-cell元素
  • vertical-align和line-height
  • 1、vertical-align的百分比是相对line-height的值来计算的
  • 
		{
			line-height: 30px;
			vertical-align: -10%;
		}
		等同于
		{
			line-height: 30px;
			vertical-align: -3px;
		}
  • 2、vertical-align默认值是baseline，所以会出现下图的情况：解决办法：
    A、<img>添加样式display:block。
    B、改变vertica的值，不要让它默认baseline对齐。
    C、把line-height的值设为0或font-size值设为0。
