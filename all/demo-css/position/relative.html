• relative限制作用
  • 1、限制absolute定位范围
  • 2、限制absolute和fixed的z-index层级
  • 3、限制absolute不再可以超越overflow:hidden
  • relative定位
  • 1、相对自身，就是top:0;left:0;位置不会发生变化。
  • 2、无侵害，就是不会响应其他元素布局。
  • 3、同时设置top、bottom只有top生效，同时设置left、right只有left生效。
  • relative和层级
  • 1、relative可以提高元素的层级。
  • 2、一个元素只设置了relative而没有设置z-index或者设置了z-index:auto，这个元素就不会限制absolute和fixed的z-index层级（IE8+）
