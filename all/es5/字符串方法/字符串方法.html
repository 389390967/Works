• charAt()
  • charAt() 方法可返回指定位置的字符
  •     	var str = '字符串';
    	console.log(str.charAt(0));
    	console.log(str.charAt(1));
    	console.log(str.charAt(2));
    	输出：字
    	输出：符
    	输出：串
  • 注释：字符串中第一个字符的下标是 0。如果参数 index 不在 0 与 string.length 之间，该方法将返回一个空字符串
  • charCodeAt()
  • charCodeAt() 方法可返回指定位置的字符的 Unicode 编码。这个返回值是 0 - 65535 之间的整数
  •     	var str = '09';
    	console.log(str.charCodeAt() + '-' + str.charCodeAt(1));
    	输出：48-57

    	var str = 'AZ';
    	console.log(str.charCodeAt() + '-' + str.charCodeAt(1));
    	输出：65-90

    	var str = 'az';
    	console.log(str.charCodeAt() + '-' + str.charCodeAt(1));
    	输出：97-122
  • 注释：字符串中第一个字符的下标是 0。如果参数 index 不在 0 与 string.length 之间，该方法将返回一个NaN
  • fromCharCode()
  • fromCharCode() 可接受一个指定的 Unicode 值，然后返回一个字符串
  •     	console.log(String.fromCharCode(48));
    	console.log(String.fromCharCode(49));
    	console.log(String.fromCharCode(50));
    	输出：0
    	输出：1
    	输出：2
  • 注释：该方法是 String 的静态方法，字符串中的每个字符都由单独的数字 Unicode 编码指定。
    它不能作为您已创建的 String 对象的方法来使用。因此它的语法应该是 String.fromCharCode()，而不是 myStringObject.fromCharCode()。
  • substring()
  • substring() 方法用于提取字符串中介于两个指定下标之间的字符
  • stringObject.substring(start, stop)
    1、参数：start（必需）一个非负的整数，规定从哪里开始截取字符串。
    2、参数：stop（可选）一个非负的整数，规定从哪里结束截取字符串（比要被提取的最后一个字符的位置+1）。如果省略该参数，那么返回的子串会一直到字符串的结尾。
  • 注释：
    1、substring() 方法返回的子串包括 start 处的字符，但不包括 stop 处的字符。
    2、如果参数 start 与 stop 相等，那么该方法返回的就是一个空串（即长度为 0 的字符串）。
    3、如果参数 start 比 stop 大，那么该方法在提取子串之前会先交换这两个参数。
    4、与 slice() 和 substr() 方法不同的是，substring() 不接受负的参数。（负参数当做0来处理） 
  •         {
            var str = '零一二三四五六七八九';

            console.log(str.substring(0, 10));
            输出：零一二三四五六七八九
            console.log(str.substring(5, 5));
            输出：
            console.log(str.substring(10, 0));
            输出：零一二三四五六七八九
            console.log(str.substring(-2, 2));
            输出：零一
        }
  • slice()
  • slice() 方法可提取字符串的某个部分，并以新的字符串返回被提取的部分
  • stringObject.slice(start, stop)
    1、参数：start（必需）规定从哪里开始截取字符串。如果是负数，则从字符串的尾部开始算起。也就是说 -1 就是最后一个字符
    2、参数：stop（可选）规定从哪里结束截取字符串（比要被提取的最后一个字符的位置+1/-1）。如果省略该参数，那么返回的子串会一直到字符串的结尾/开始。
  • 注释：
    1、如果参数 start 比 stop 大，那么该方法在提取子串之前不会先交换这两个参数,返回的是一个空串。 
  •         {
            var str = '零一二三四五六七八九';

            console.log(str.slice(-3, -1));
            输出：七八
            console.log(str.slice(8));
            输出：八九
            console.log(str.slice(-3));
            输出：七八九
            console.log(str.slice(3, 1));
            输出：
            console.log(str.slice(-1, -3));
            输出：
        }
