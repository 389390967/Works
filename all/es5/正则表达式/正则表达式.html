• 规则表达式就是用于描述规则的工具。换句话说，规则表达式就是记录文本规则的代码。
  • 常用元字符
    \w (word)匹配字母或数字或下划线[a-z0-9_]
    \s (space)匹配任意的空白符
    \d (digital)匹配数字[0-9]
    \b 匹配单词的开始或结束(单词边界)
    . 匹配除换行符以外的任意字符
    ^ 匹配字符串的开始
    $ 匹配字符串的结束
    常用反义词
    \W 匹配任意不是字母，数字，下划线，汉字的字符[^a-z0-9_]
    \S 匹配任意不是空白符的字符
    \D 匹配任意非数字的字符[^0-9]
    \B 匹配不是单词开头或结束的位置
    [^a] 匹配除了a以外的任意字符
    [^asdfg] 匹配除了asdfg这几个字母以外的任意字符
    常用限定符
    * 重复零次或更多次{0,}
    + 重复一次或更多次{1,}
    ? 重复零次或一次{0,1}
    {n} 重复n次
    {n,} 重复n次或更多次
    {n,m} 重复n到m次
    匹配选项
    i ignore 忽略大小写匹配
    g global 全局匹配

  • 正则和match()配合使用
  •     	var re = /\d/g;
    	var str = 'asd e123 fdsa 456 789gg ethan';
    	console.log(str.match(re));
    	输出：["1", "2", "3", "4", "5", "6", "7", "8", "9", "1", "2", "3", "4", "5", "6", "7", "8", "9", "1", "5", "9", "1", "1", "9", "9", "8", "8"]

    	var re = /\d\d/g;
    	var str = 'asd e123 fdsa 456 789gg 123456789 ethan 159 11 99 88';
    	console.log(str.match(re));
    	输出：["12", "45", "78", "12", "34", "56", "78", "15", "11", "99", "88"]

    	var re = /\d+/g;
    	var str = '0 0 asd e123 fdsa 456 789gg 123456789 ethan 159 11 99 88';
    	console.log(str.match(re));
    	输出：["0", "0", "123", "456", "789", "123456789", "159", "11", "99", "88"]

	去空格
	var str = '  aa bb  cc   dd  ';
	console.log(str.replace(/\s+/g, ''));
	输出：aabbccdd

	去首尾空格
	var re = /^\s+|\s+$/g;
	var str = 'aa bb cc dd ';
	console.log(str.replace(re, ''));
	输出：aa bb cc dd

	检验中文
	var str = '哈哈 asdfgqwer 12 !@#$ *&^ +_=';
	var re = /[\u4e00-\u9fa5]/;
	console.log(re.test(str));
	输出：true

	var str = '哈哈 asdfgqwer 12 !@#$ *&^ +_=';
	var re = /^[\u4e00-\u9fa5]$/; 加了^ $以后表示行首和行尾之间必须出现中文而且只能出现中文
	console.log(re.test(str));
	输出：false

	检验QQ
	var re = /[1-9]\d{4,10}/g;
	var str = 'QQ：159753，QQ：159753，QQ：0-14753';
	console.log(str.match(re));
	输出：["159753", "159753", "14753"]

	检验邮箱
	var re = /^\w+@[a-z0-9]+\.[a-z]{2,4}$/;
	var str = '159753@qq.com';
	console.log(re.test(str));
	输出：true
分析方法————以邮箱为例
一串字母、数字、下划线 @ 一串字母、数字 . 一串字母（最少2个、最多4个）
\w+ @ [a-z][0-9]+ . [a-z]{2,4}
/\w+@[a-z0-9]+\.[a-z]{2,4}/
